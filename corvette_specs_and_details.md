**Obsolescence notes (why some 1984 specs need an override):**
- **Engine oil — API "SF" is obsolete.** Current oils are API SP. The bigger issue for this engine: the L83 has a **flat-tappet camshaft**, and modern API SP oils have reduced anti-wear zinc/phosphorus (ZDDP). Many owners of flat-tappet GM V8s run an oil formulated with higher ZDDP (classic/"hot rod" oils, or oils marketed for older engines) or add a ZDDP supplement. Pick what you actually run and record it here.
- **DEXRON-II ATF is discontinued.** GM's licensed replacement is **DEXRON-VI**, which is backward compatible and is the usual modern substitute for both the **overdrive unit** and any DEXRON-II application. (Some prefer a quality multi-vehicle ATF; record your choice.)
- **GM-specific part numbers** (coolant 1052753, brake fluid 1052535, PS fluid 1050017, chassis grease 1051344, gear lube 1052271, LS additive 1052358, washer solvent 1051515, etc.) were dealer items; most are superseded. Equivalents are widely available — **DOT-3 brake fluid, conventional green ethylene-glycol coolant, 80W-90 GL-5 gear lube, lithium chassis grease, and a limited-slip friction modifier** all still exist off the shelf. None of those base specs are themselves "wrong," just repackaged under modern part numbers.

*These modern-equivalent suggestions reflect common practice for this era of car, not a professional service recommendation — verify against a current service source or your shop before filling.*

**Key fluid points for this car (manual RWD Corvette, Doug Nash 4+3, limited-slip):**
- **Transmission main 4-speed box** → **SAE-80W or 80W-90 GL-5 gear lubricant**.
- **Overdrive unit** (the "+3" of the 4+3) → **DEXRON-II ATF** (modern override: DEXRON-VI). *(The two halves of the Doug Nash take different fluids — matches the separate capacities listed above: 4-speed box ~2 pt, OD unit ~4 pt.)*
- **Rear axle** → limited-slip gear lube **GM 1052271**, and add **4 oz of limited-slip additive (GM 1052358)** before filling, or it will chatter.
- **Clutch linkage / floor shift linkage** → engine oil (manual-car items).
